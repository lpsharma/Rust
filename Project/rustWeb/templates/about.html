About rustWeb

A full-stack web application written in Rust, serving as a learning reference for how web apps are structured.

Tech Stack

🌐
axum
Web framework built on hyper + tokio. Ultra-fast, ergonomic routing and middleware.
🎨
tera
Jinja2-inspired template engine. Renders HTML server-side with Rust data.
📦
serde
Serialization framework. Converts Rust structs ↔ JSON effortlessly.
⚡
tokio
Async runtime. Handles thousands of concurrent connections with minimal overhead.
🔒
tower-http
Middleware: CORS, static file serving, request tracing.
🆔
uuid + chrono
UUID v4 for unique IDs. Chrono for timestamps with serde support.

Rust Patterns Used in This App

  • Arc<AppState> — share state across async handlers safely
  • Mutex<Vec<T>> — in-memory database with concurrent access protection
  • #[derive(Serialize, Deserialize)] — automatic JSON marshaling
  • Result<T, E> — all API handlers return errors explicitly
  • async fn — every handler is an async function powered by tokio
  • Traits (IntoResponse) — axum handlers can return any type implementing IntoResponse
  • Extractors (State, Path, Json) — axum pulls typed data from requests automatically
